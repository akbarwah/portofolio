# Professional Portfolio - Akbar Wahyu

This is my professional portfolio website built with **Next.js**, **Tailwind CSS**, and **Framer Motion**.

## Features
- Responsive Design
- Smooth Animations
- Contact Form Integration
- Psychology-Driven HR Content

## Tech Stack
- Framework: Next.js 15
- Styling: Tailwind CSS
- Icons: Lucide React
- Deployment: Vercel
